# Facial Emotion Recognition using MobileNetV2

## Description
This is a computer vision project built in MATLAB that identifies facial emotions from images. The project utilizes a transfer learning approach, adapting a pre-trained MobileNetV2 model to classify different human emotions. It includes a training pipeline with data augmentation and a user-friendly Graphical User Interface (GUI) for easy image testing.

## Prerequisites
To run this project, you will need:
* MATLAB (R2020a or later recommended)
* Deep Learning Toolbox
* Deep Learning Toolbox Model for MobileNet-v2 Network

## Project Structure
* **`ProjectModel.m`:** The script used to load the image dataset, apply data augmentations, replace the top layers of MobileNetV2, and train the model using the Adam optimizer.
* **`ProjectGUI.m`:** The interface script that allows users to upload images and view the predicted emotion.
* **`emotion_model_mobilenetv2.mat`:** The trained neural network model generated by the training script.

## How to Use

### 1. Training the Model
1.  Open `ProjectModel.m`.
2.  Update the `datasetPath` variable to point to your local dataset directory. Ensure your dataset is organized into subfolders, where each subfolder name represents the emotion label.
3.  Run the script. The model will train for 10 epochs and save the resulting network as `emotion_model_mobilenetv2.mat`.

### 2. Running the GUI
1.  Ensure the `emotion_model_mobilenetv2.mat` file is in your current working directory.
2.  Run `ProjectGUI.m`.
3.  Click the **📷 Upload Image** button to select an image from your computer.
4.  The GUI will process the image (converting to RGB if necessary and resizing) and display the predicted emotion on the screen.
